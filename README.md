# Duck_Hunt_pygame
Learning project to recreate/remaster Nintendo's NES game Duck Hunt on python with pygame module.
This is my first ever project 

##Preview


## Features
- Main menu with "Start" and "Exit" and options highlight 
- Moving duck which bounces at the window edges 
- Ammo and reload system
- crosshair which changes color when on the duck 
- Duck animation for flying and falling 
- Dog animation for round intros, presenting duck and laughing on miss 
- Round progression with increasing difficulty
- Score and high score tracking
- Sound effects and music

## Controls 
Selecting in menu-Left mouse button
Aim-Mouse movement
Shooting-Left mouse button

## Installation 
(Method 1)
1. Download the attached file "Duck_Hunt_v1.0.0.zip"
2. Extract it
3. Run the exe

(Method 2)
1.Install python latest version from https://www.python.org/
2.Install pygame with "pip install pygame" command in terminal 
3.Download the code from main brach 
4.Open the location of duckhunt.py with terminal and run it with command "python duckhunt.py "
